# Student Performance Analysis using Deep Learning

## Overview
This project predicts student academic performance using Deep Learning models. It uses student academic datasets to analyze learning patterns and predict performance.

## Technologies Used
- Python
- TensorFlow
- Keras
- Pandas
- NumPy
- Scikit-learn

## Dataset
- student-mat.csv
- student-por.csv

## Features
- Student performance prediction
- Data preprocessing
- Deep Learning model training
- Model evaluation
- Performance comparison

## Project Files
- app.py
- train_model.py
- requirements.txt
- student-mat.csv
- student-por.csv
- model_comparison.csv
- dl_model_comparison.csv

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python train_model.py
```

## Output
- Trained Deep Learning model
- Student performance prediction
- Model comparison results
